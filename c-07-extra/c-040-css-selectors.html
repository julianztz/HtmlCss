Excercise:

  • use 'tag selector', set table border with style 'solid 2px lightblue'
  • use 'multiple selector', set rows background-color 'skyblue'
  • use 'direct child selector', set rows (except the table head row) background-color 'white'
  • use 'class selector', set cell with number >= 600 font color 'navy'
  • use 'id selector', set a number you like with color 'brown'
  • use 'pesudo-class selector', set the 3rd column with background-color 'pink'
  • use :nth-child(2)
