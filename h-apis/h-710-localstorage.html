Cookie vs Local storage

  • Cookie is sent to server for every request
  • Cookie has about 4k size limit
  • Cookie can be expired

Check local storage in Chrome devtools

Resources->Local Storage

How to clean local storage?

localStorage.clear()

Local storage usages

  • set: localStorage.experience="teens programming"
  • set: localStorage['experience']="teens programming"
  • get: localStorage.experience
  • delete: localStorage.removeItem('experience')
  • delete: delete localStorage.experience
  • localStorage.clear()
